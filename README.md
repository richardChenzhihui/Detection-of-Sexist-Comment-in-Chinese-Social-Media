# Detection-of-Sexist-Comment-in-Chinese-Social-Media
Sexist content on social media has been a widely discussed public issue, but there have been few relevant studies conducted by the Chinese NLP communities. In our project for the HKU ARIN7101 "Statistics in Artificial Intelligence" course, we, the group members (listed alphabetically) **Chen Zhihui, Fan Yishuang, Fang Cunling, Guo Minxin** and **Sun Yize**  proposed an SVM model with RBF kernel based on word2vec sentence embedding trained on a customized corpus and PCA to conduct semantic classification on text collected from Chinese social media.
